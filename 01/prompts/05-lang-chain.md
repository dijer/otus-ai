## Инициализация lang chain

```text
создай в проект папку для агента lang chain на nodejs + ts.
и добавь .env.example. 
```


## Установка зависимостей

```text
добавь зависимости для langchain агента.
в .env.example добавь примеры для провайдера.
секреты не коммитить.
```


## Реализация Tools

```text
реализуй langchain tools для реального http вызова апи:
getQuestons - GET /questions
submitAnswers - POST /answer

верни структурированный результат как объект json
```


## Системный промпт

```text
добавь системный промпт для роли апи оператора.
агент работает только через tools, не выдумывает данные, при ошибке пишет причину.
контракт:
status: success || error
action: описание действия
data: результат api
errors*: список ошибок
```


## Запуск

```text
сделай запуск из cli npm run dev -- "создай ответы для анкеты: имя Alex, возраст 25, любимый ai GPT"
аргумент из командной строки передавай агенту как user input.
если аргумент пустой - выведи подсказку
```
