## Промпт: подготовка AI-инструкций для репозитория

Цель:
Подготовить проект к работе с AI-инструментами так, чтобы агент быстрее входил в контекст, действовал в заданных границах и выдавал воспроизводимый результат.

Задача:
1. Добавить репозиторные инструкции для GitHub Copilot.
2. Добавить документы контекста, архитектуры, стиля, Definition of Done и типовых задач.
3. Включить проверочные требования, по которым легко увидеть соблюдение инструкций.
4. Сохранить артефакты в markdown-файлах внутри репозитория.
