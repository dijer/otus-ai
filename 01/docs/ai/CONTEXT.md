# CONTEXT

## Что это за проект

"Минианкета" - учебное fullstack-приложение с локальным backend (Go) и frontend (React + Vite).
Пользователь открывает страницу, видит список вопросов анкеты, заполняет ответы и отправляет форму.

Основной пользовательский сценарий:
1. Frontend запрашивает вопросы у backend.
2. Пользователь заполняет поля формы.
3. Frontend отправляет ответы на backend.
4. Backend сохраняет ответы в памяти процесса и подтверждает успех.

## Как запустить локально

Backend:
1. Перейти в `backend/`
2. Выполнить `go run main.go`
3. Проверить `http://localhost:8080/questions`

Frontend:
1. Перейти в `frontend/`
2. Выполнить `npm install`
3. Выполнить `npm run dev`
4. Открыть `http://localhost:5173`

## Границы

- Хранилище ответов - in-memory (внутри процесса backend).
- Нет авторизации, БД, миграций, внешних интеграций.
- Проект ориентирован на простые и проверяемые изменения.
