# DECISIONS

## Зафиксированные решения

1. Хранение ответов - только в памяти процесса backend.
2. Контракты `GET /questions` и `POST /answers` считаются стабильными для учебного проекта.
3. CORS ограничен origin `http://localhost:5173` для локальной разработки.
4. Основной AI-инструмент в проекте: GitHub Copilot Chat (VS Code), источник правил: `.github/copilot-instructions.md`.

## Что можно пересмотреть

- Добавление персистентного слоя (БД).
- Расширение API (новые endpoint).
- Более строгая валидация payload.
