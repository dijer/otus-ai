# DEFINITION_OF_DONE

Задача считается выполненной, если:

1. Изменения ограничены требуемой областью.
2. Не нарушены API-контракты `GET /questions` и `POST /answers`.
3. Код собирается для затронутой части проекта.
4. Для изменений указан способ проверки (команды + ожидаемый результат).
5. Ответ агента содержит:
   - план действий;
   - список измененных файлов;
   - краткое обоснование;
   - раздел проверки;
   - строку `AI-CHECK: OTUS-AI-01`.
